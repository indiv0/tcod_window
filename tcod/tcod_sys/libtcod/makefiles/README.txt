*******************
Linux prerequisites
*******************
To be able to compile libtcod on Linux, make sure that following packages are installed :
* gcc 
* g++ 
* make 
* zlib 
* libpng12 
* libsdl1.2 
* upx 
* electric-fence
* libpng12-dev 
* libsdl1.2-dev
